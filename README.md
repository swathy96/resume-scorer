# Resume + JD Fit Scorer

An AI-powered tool that scores how well a resume matches a specific job description. Uses the Claude API to generate a structured fit score, flag missing ATS keywords, identify gaps, and suggest rewrites for weak resume bullets.

## Why I built this

During my job search, I was scoring my own resume against job descriptions manually — comparing keywords, identifying weak bullets, figuring out skill gaps. It's an obvious automation candidate and a useful showcase of production AI integration patterns: structured outputs, schema validation, and careful prompt engineering.

## Stack

- **Backend:** Node.js, Express, TypeScript, Anthropic SDK
- **Frontend:** React, TypeScript, Vite (coming soon)
- **AI:** Claude Sonnet 4.5
- **Validation:** Zod for runtime schema checking
